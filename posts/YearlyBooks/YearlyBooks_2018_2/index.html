Comment

Jul

《怦然心动的人生整理魔法》
[日本] 近藤麻理惠 / 徐明中 / 译林出版社 / 2012
7.10 心动。简单的原则，经过解释，就成了魔法。一一面对，消化与物品的关系。小小的事，一点点建立自信。一切的方法都在每天的行动里。

《一个人的花园》
谢倩霓 / 少年儿童出版社 / 2015
7.10 能读出刻意平淡化的书写，但有些不伦不类

《来自液态现代世界的44封信》
[英国] 齐格蒙·鲍曼 / 鲍磊 / 漓江出版社 / 2013
7.11 前几篇和后几篇好。翻译垃圾。液态世界，选择与拒绝的权力被交还于人们自己。认真生活。自行界定世界。

《怦然心动的人生整理魔法2》
[日本]近藤麻理惠 / 颜尚吟 / 译林出版社 / 2014
7.15 就像被占了大量内存运行缓慢的电脑，物品若占据了家中太多的空间，人也会变得迟钝犹疑。这本对1中的原则做了更多更细致的解释，能理解得更深入些。很快就能翻完。把卧室整理了一番后，心情确实有变化。让喜爱的事物可见和常见，以便常常惦念，就像我把一些灵感放在待办事项里，闲下来时复习，召回当时的感觉再继续思考下去。

《爱的缓刑》
[法国] 让-皮埃尔·吉布拉 / 杜江鑫、方堃 / 湖南美术出版社 / 2018
7.22 名字能简单地翻成《缓刑》就更好了。更像一篇散文，以一个不存在的第三视角呈现，这些日记样、反故事的琐碎记录。因了最后几页发生的事，才终于有了名字，没有结束的结尾。

《榛子的味道》
[法国] 瓦妮妲 / 吕俊君 / 后浪丨北京联合出版公司 / 2018
7.22 遇见与走失、珍视与遗忘、摇曳瞬间、哭泣、尴尬、飘忽不稳、一夜之情、缠绵、辗转、仰慕、孤独、不属于任何人、失误与犯错、死别、重逢。人总是在情爱里见到、感到、学到许多许多，也更懂得了关照自己。

Aug

《控制论与科学方法论》
金观涛、华国凡 / 新星出版社 / 2005
8.6 控制论，认识论，系统论都是我的盲区，这本足够深入浅出，能串连起很多零散的想法。尤其是系统论、飞跃和渐变、蝴蝶型突变、中间路标。需要把维纳翻出来研读一下，觉得这是现代社会的一个本质要素。

《Henri Matisse Jazz》
[法国] Henri Matisse / Wiethege, Katrin / 2009
8.10

《东京一年》
蒋方舟 / 中信出版社 / 2017
8.12 中信出的书还是噱头优先。前1/3是不错的。感觉删减了很多，也续貂了很多。若是去掉照片和那些小说梗概，不知道还剩下多少东西。引用做得不错，但阐发的部分就弱了很多，以至于很多想法写到最后变得幼稚而俗套。不过她至少保留了一份属于自己的清醒。

《初期短篇集》
[日本] 谷崎润一郎 / 陈若雷 / 广西师范大学出版社 / 2018
8.12 日本女性的符号感，看了三篇后后才进入状态。虽混乱，但丰盛。

《少年绿皮书 : 我们的岛屿旅行》
刘克襄 / 上海译文出版社 / 2015
8.18 台中市乌日区。微小的世界，慢慢变大，一个监狱一个操场一条逃学的路线。自然的巨大、信息、广阔，无尽的名字。不同代际邂逅的不同的老书店。一场期待的旅行。邹族人，阿里山。几篇接续的散文，几幅插图，写出了伴随成长，世界地图不断变得宽广的历程。我们不知怎的降临，又手足无措地成长，经历的一切都渐渐融入了自我。监狱，学校，操场，大树，逃学的路线，山与森林，最后才出现了城市。

《圈外編輯》
[日本] 都築響一 / 黃鴻硯 / 臉譜 / 2018
8.20 很喜欢他在一席的演讲。一个近乎孤军奋战着的人，六十岁，感叹自己身处有趣的时代。圈外、报导、记录、传达，脚下的土地，随处可见的生活，视而不见的隐忧，他惦念的主题，都格外直白且坦诚。敏感。亲历派。电邮形式的付费电子杂志，也想试试。

《念书，还是工作？ : 这是一个问题》
[法国] 蒂菲娜·里维埃尔 / 潘霓 / 拜德雅丨西南师范大学出版社 / 2018
8.20 “当你不再放弃往上攀登，脚步才不会停下”。核心并非外界的矛盾，而是自我的转向。

《本托的素描簿》
[英国] 约翰·伯格 / 黄华侨 / 理想国 | 广西师大出版社 / 2017
8.22 本体，自生，身体的本质即心灵的永恒，不存在限制。竟然从素描出发得到这样的感悟。不可见的事物与不可预知的终点，由概念和观念出发，确凿无疑而又无影无形。通过故事学会如何给予关注，继承并创造。拯救此刻，幸运者们对当下施以援手。自然充斥着行文，伯格拥有令人艳羡的日常。

《抵抗的群体》
[英国] 约翰·伯格 / 何佩桦 / 理想国 | 中国美术学院出版社 / 2018
8.26 不可见。相似性。内在经验。反抗。内源。此刻。放在一起已经相当庞杂。跟着伯格能够更好地学会看，看不在场的一切，或期待或目睹它的到来。

Sep

《人有人的用处 : 控制论和社会》
[美国] N·维纳 / 陈步 / 商务印书馆 / 1978
9.3 确实是哲学书籍。信息传输与熵增的不可违背。延展至法律、社会、工业革命、机器演化之后，维纳又提及神话，以及提议科学家应恪守其“纯朴”。很多都与塔勒布的“非对称性”观点有很强的联系。「法律责任不应当是这样的性质：一方被迫行动，而另一方自由。」忠实、认真、信仰、朴实，让我又想起了 Ihalmuit 们。

《这不是书》
[法国] 让·朱利安、Jean Jullien / 浪花朵朵童书 编译 / 北京联合出版公司·后浪出版公司 / 2016
9.7 会心一笑

《蓝色小药丸》
[瑞士] 弗雷德里克·佩特斯 / 陈帅、易立 / 后浪丨北京联合出版公司 / 2017 绘本
9.7 概念对人的异化。沉重和轻松的交替。活生生的频繁的日常生活。“别要求事情如你想象的那样发展，而要顺着事情发展的方向去期望”。西方人一般在化解矛盾和冲突中成长，厄运与机会同在。

《什么是批判？/ 自我的文化 : 福柯的两次演讲及问答录》
[法国] 米歇尔·福柯 / 潘培庆 / 拜德雅丨重庆大学出版社 / 2017
9.8 批判是对某一领域的关注。自我的实践，就是个人再和他自己的关系中把自己构成为主体的方法。主体在具体的实践中，在历史上可分析的实践中构成自己。通过“记事本”来内化规则与自我技术，完善自我叙事。福柯的演讲，展示了主题的确定和验证过程、挖掘的条理和思路，不停质疑反诘，并接受质疑，连接了他几部主要作品的潜在思路，很有价值的文本补充。

《什么让这个世界转动》
[西班牙] 科尔曼·乌里韦 / 罗秀 / 漓江出版社 / 2015
9.12 是爱。是英雄的爱。是被无情时代吞没了的好心的普通人。数不尽的普通英雄的死亡，口头传说里有他们的记忆和故事，这是世界运转下去的理由。无形的墓碑。幸存者得了英雄的魂。

《HAPPINESS》
[日本] 押見修造 / 林志昌 / 東立出版社 / 2018
01 9.22
02 9.22
03 9.22
04 9.22 还是这个风格，心理走向诡异，眨眼多年。

《农场 : 从印尼到墨西哥，一段直击动物生活实况的震撼之旅》
[美国] 索尼娅·法乐琪 / 范尧宽、曹嬿恒 / 中国政法大学出版社 / 2018
9.23 原名为《PROJECT ANIMAL FARM》，是一个作者自发的调查案的集录。部分内容通过纪录片有过了解。快速盈利的目的，决定了这样的工业化农场结构。进入都市的反面，一味规模化的反面，虽是闯入，却目击了人性。作为自我的开端，是探索，定义节奏，脱离体系性风险；作为人的开端，是成为有益的讯息。

《符号帝国》
[法国] 罗兰·巴尔特 / 汤明洁 / 中国人民大学出版社 / 2018
9.24 挺薄，《包装》和《内在与外在》两篇最惊喜。用一套在巴特心中即将崩塌的符号和意义体系，来诠释日本，有趣且过瘾。核心是“空”，体系是“禅”。执拗于框限、过程、动作、工作，于是产生匠人。空无中心的城市，分成无数的丁目、番地，块状街区，是活生生的另一种系统性。万物被“空”的框限所框限，被“空”的符号所承载和传送。木偶戏打破二元关系，消灭内在与外在的传递。禅宗式地只抽取一种表达方式，执着于（框限的、空的）符号和形式，放弃意义。描述和定义都在俳句中消失。盘绕自身，优雅的环路，即为日式的自身与自我。想起那句“人即信息”。「毗邻奇遇的灰暗之地」

Oct

《被禁锢的头脑》
[波兰] 切斯瓦夫·米沃什 / 乌兰 易丽君 / 广西师范大学出版社 / 2016
10.3 恐惧与脆弱之时的穆尔提-丙药丸。唯物辩证法诡辩术与权力话语。知识被创造并灌输。废墟与痛苦是社会思想的学校。米沃什不断挖掘记忆，对比着剖析自己，自述自己的来路，找到天赋所赋予的使命。阻力和障碍对创作的所谓助益，在于亲身经历，在于对人的悲悯。在这历史潮流中需耗费极大的气力以保持自我，保留人性的种子。实践抗议。

《开箱整理我的藏书 : 本雅明读书随笔》
[德国] 本雅明 / 金城出版社 / 2014
10.7 回忆的过程。金宇澄说作家就是在记录那些无用的素材，普鲁斯特把它们编织成了世界，发扬光大了“幸福意志”。这是欧洲人的骄傲所在吧。“词语从来不在源头”。“我的全部错误都为了率先落在后面”。忘却，是一个完美的容器。最远的异乡是自己，这是卡夫卡一百年前关注到的。灵魂、眼睛、手。现代人不再做不可缩略之事了。灵晕在震惊经验中分崩离析。

《学会完结》
[法国] 罗朗·莫维尼埃 / 余中先 / 湖南文艺出版社 / 2017
10.11 辗转难眠的自我叙述。

《态度 : 吴军家书》
吴军 / 中信出版社 / 2018
10.30 枫蓝中信书店翻完。多数信甚至都附带了缘起和结果，读来意思就是：“你就该这么办”！不知道是编辑的意思还是作者的意思。用的都是激励的方式，夸奖为主，指导为辅。每封信后的“你的父亲”蛮刺眼。做笨人、投资方法、科学思维几篇蛮好。

Nov

《文艺杂谈》
[法国] 保罗·瓦莱里 / 段映虹 / 生活·读书·新知三联书店 / 2017
11.7 想读全本啊。思考的体量大得可怕。只是都纠缠在一起，没有更深入、细致的梳理。若不是演讲逼稿，恐怕难以公开。每一个问题对他来说，或许都是极宏大的，只能远观，尝试接近，更未曾想过抵达。看到很多后来的哲学家的理论影子，源流究竟怎样，无从知晓了。

《索福克勒斯悲剧五种》
[古希腊] 索福克勒斯 / 罗念生 / 世纪文景/上海人民出版社 / 2016
11.25 「他不但不要朋友领路，反而给我们大家领路。」直观、有力的叙述，没有分毫停顿。对话与言语，亦不必事无巨细。逃避还是接受，昭示着两种命运。

《王佐良全集•第八卷》
王佐良 / 外语教学与研究出版社 人文社科工作室 / 2016
11.26 要读穆旦和戴望舒的译诗。
