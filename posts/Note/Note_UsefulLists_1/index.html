有价值的清单

公共决策备忘清单

  1. 促进正义；
  2. 修复互惠性；
  3. 协商可划分与不可划分的收益；
  4. 考虑人民的利益多过机器的利益；
  5. 其策略是扩大收益还是减小祸患；
  6. 节约多过浪费；
  7. 可逆行动多过不可逆行动。

《技术的真相》 P179

关于一本书，一定要提出四个主要的问题

  1. 整体来说，这本书到底在谈些什么？
  2. 作者细部说了什么，怎么说的？
  3. 这本书说得有道理吗？是全部有道理，还是部分有道理？
  4. 这本书跟你有什么关系？

《如何阅读一本书》

做笔记的几种可采用的方法

  1. 画底线
  2. 在画底线处的栏外再加画一道线（突出记号）
  3. 在空白处做星号或其他符号（慎用，可以折角替代）
  4. 在空白处编号
  5. 在空白处记下其他的页码（关联书的其他部分）
  6. 将关键词或句子圈出来
  7. 在书页的空白处做笔记

另：书后的空白页可以用来作为个人的索引页，记录作者的主要观点；书前的空白页最好是用来记载你的思想，表现出你在智力上对这本书的所有权。

《如何阅读一本书》

主题阅读的五个步骤

前提：已检视相当多的书，对其中一些书在谈什么有了概念，且有了想要研究的主题

  1. 找到相关的章节（你及你关心的主题才是基本的重点，而不是你阅读的书，且这可能与作者的写作目的相去甚远）
  2. 带引作者与你达成共识（强迫作者使用你的语言，主题阅读是一种大量的翻译工作）
  3. 厘清问题（由自己建立词汇，并建立起一组不偏不倚的主旨，梳理概念）
  4. 界定议题（围绕主题主旨的争论）
  5. 分析讨论（突破争论的困境）

《如何阅读一本书》

避免污化心智程序的策略

  1. 避免安装可能对你产生生理伤害的心智程序（如饭后一根烟）
  2. 关注影响你目标的心智程序，确保它不妨碍目标选择的多样性（如异教，有整套强制准则的社群或公司）
  3. 关注同认识世界相关的信念和模型，寻找真实反映世界的心智程序（自行建构，如自学）
  4. 警惕拒绝对自身进行评估的心智程序（经常重新复盘及评估，撰写自己的行为模式及方式清单，并不断检验、剔除、修正、完善）
