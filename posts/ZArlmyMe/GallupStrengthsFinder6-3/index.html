盖洛普优势识别系列之三：搜集

Jan 11, 2017 | 盖洛普优势识别器 | 阅读 | 本文共计 1.6k 字 | 阅读约需 5 分钟

什么是 Input

Input 就是无法抑制的好奇心。在他们眼中，世间万物都是可收集的对象，只因觉得：它们可能会有用。

只要是符合自己的想法、理念和喜好的东西，就统统都想保存下来。

结合个人感受说说 Input

看到测试结果中的“搜集”时，感觉自己是被砸中的。

Input，只用一个词来形容，一定是“停不下来”。一旦挖下了坑，后果不堪设想。Youtube 视频中提到的关键词里，我个人觉得最贴切的有两个：Sources、Tools。收集控、资源控，也是我乐于标榜自己的标签，坚决不容许自己在资源上居于弱势，找不到想要的东西，一定是对 Input 最大的折磨。Input 可能不是新世界的发现者，而如果搭配上有发散性的天赋，就一定会变成八爪鱼，满世界乱转，随着时间积累，渐渐变得“万能”。

“万一有用呢”、“万一哪天就找不到了呢” 是我管不住手时最常用的说辞。对于下载的狂热，源自于多次数据丢失的经历；对实体产品的由衷热爱，也是因为：它们往往比电子的产品可靠得多、安全得多。我还有强烈的恋旧情结，但凡承载着记忆的东西都不忍舍弃，这算是一种对信息载体的痴迷，中介物、介质、媒体、聚合，都是令我痴迷的词汇。这或许是所谓『断舍离』和『极简主义』的反面，这些“物”不断印证着你的存在，同时也成为关于你的“索引”。我觉得高级的创造者绝不会提倡极简，而应诉诸于更高级的管理体系或整理方法。

另一方面，也可能是我害怕失去，常常对未知心生恐惧，就如 Stark 家族的族语『Winter is Coming』，这成为了一种有关生存的意志。像秋天时的松鼠，不停用身体储存能量的同时，还会藏匿超量的越冬食物（主要是种子）。其中一些会被挖出来吃掉，而被遗忘的，在次年的春天萌发。无形中，松鼠成了生态链中的重要一环。

在互联网上，我偏好进行大量的浏览和搜索，搜索引擎给出的结果，我也不只看最优答案。对于自己未知的领域，甚至把前几页逐一看过去（遍历）。这基于一种“强迫症”心态：有些信息，永远潜藏在背后，只有通透过大量的阅读工作才能找到些许踪影。

在生活里，我对居住的街道了如指掌，对于城市中自己感兴趣的场所和地点如数家珍（比如看演出、展览，参与活动，以及吃吃吃）。我一般只需要两个星期，就能搞清楚在一座新的城市里，应该怎么玩，在探路之后确认有哪些落了俗套，有哪些值得一去，从而拥有自己的一套玩法。

我的桌面每年能保持整洁的天数，可能不超过10天。看到文首的图片你一定能懂我在说什么，但我的背包里却不会这样凌乱不堪。或者可以这样界定：跟工作台和桌面概念相关的空间，永远是凌乱的。比如笔记本电脑的桌面，比如云盘的首页。

在这方面，有 Input 的人，或许都像触手系。

遍历的好处是，短时间内摄入大量相似信息，信息之间会发生对照，加速内部淘汰。这一过程中，会有一种“因量变产生质变”的感觉，但不能忘记，最重要的是『出发点』，基于什么目的进行这一系列的探索，Input 有一种复杂化的倾向，产生的质变可能是新的灵感、新的概念，青出于蓝的一次次“涌现”，从一种具体的现实生发出略显抽象的现实，多者异也。

How you do, what you value ?

凡事以积累为导向，一旦有了主题（出发点），就会用尽一切办法，将其填满，并不断探索新的角度。成为万能的资源箱、工具箱，坚决不重复造轮子，为团队节约人力资源。

More is different，是一种思考方式，也可以理解为一种成长方式。基础概念不同，生发出的复杂系统自然不同，每个人都可以提出自己的理论，自己的架构，不必处处受限于已有的体系。

如何持续改进“搜集”，形成优势：

  • 有 Input，更要有 Output
  • Collect - Find - Share
    • 梳理收集流程，使输入有效且有价值
    • 能够存储和搜索的信息管理体系（安全、稳定），即合适的容器，及开启容器的钥匙
    • 建立自己的分享渠道
  • 要有特长领域，并完成深入积累，梳理方法（概念填充的过程）
  • 更大的词汇/概念水平，搜索范围足够广，才能突破某些无形的限制
  • 把控输入质量，阅读权威期刊、编辑精选后的作品，借助信任的力量

松鼠在互联网上留下的痕迹

  • 淘宝收藏 816 家店铺
  • 曾在点点网同时维护 8 个图片站
  • 在花瓣网有 66 个画板，11705 个采集
  • 在豆瓣网上有 1894 本书想读，693 张专辑想听，812 部电影想看
  • 目前 Chrome 收藏夹里有 2003 个书签
  • 在百度盘有 5047.64G 的各类资源
  • 曾做过一个资源站，一年时间，平均每天 20 张专辑资源的发布
  • 初中、高中时代买磁带听音乐，一度存有 300+ 磁带
  • 大学时代买 CD，一度存有 200+ CD
  • 为了定期存档自己喜欢的专辑和电影，自行刻录 DVD 100+
  • 截图癖，电影的截图不止 10000，日常 Lifeline 的截图整理也已持续 4 年，而更多截图则被打包扔在了移动硬盘的角落里
  • 虾米音乐收藏 792 首歌
