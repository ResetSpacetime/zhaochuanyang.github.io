Comment

Jan

清明幻河图
那多 / 2009 / 接力出版社
1.9 没有以前玄了，也没有以前严谨了，紧张时刻的搞笑我不喜欢

销售中的心理学
崔西 / 2007 / 中国人民大学出版社

光荣日
韩寒 / 2007 / 二十一世纪出版社

父亲的战场
章东磐 / 2009 / 山西人民出版社
1.21-1.23 中国国殇。一个个尘封已久的故事在作者的艰难走访后得以再现在眼前，尚且在世的老人和渐渐在中国消失的真实让我对中国的现代史充满了疑惑，对错在中国总可以分成三六九等，被物质的社会欲望掩盖，没有人再提起。能阅得本书是我的大幸。

冷记忆 5
[法国] 让·波德里亚 / 2009 / 南京大学出版社
1.20-1.25 容量太大我仍有许多无法理解，或许都明朗了之后才会有更深刻的体会

野火集
龍應台 / 1985 / 圓神出版社
借个条目标注《大江大海一九四九》1.23-1.25 龙应台大大地写了一个被时代忽略了的"人"字，在时代的更迭和挤压下，没有一句道歉没有一句感谢。这是时代于人的痛楚，拨开政治和历史的狭隘，追回即将被忘却的记忆。故事，总在等待，却没有人问津。他们没有抱怨这个时代，这个国家，最让我感到动心和悲楚。

大学毕业等于零
崔传桢 郑瑞强 / 2004 / 经济管理出版社
1.25 此时的大学生已经不是彼时的大学生了

I先生
[法国] 李维斯·唐德汗姆 / 2007 / 北方妇女儿童出版社
1.25 没有文字的绘本，寓意深刻。因果在现实中的表象。

O先生
[法国] 李维斯·唐德汉姆 / 2003 / 吉林文史出版社
1.26 奇怪的故事奇怪的道理让人喜爱

中国娼妓史
书奴 / 2004 / 团结出版社
1.24-1.26 眼镜还是有了一点颜色

白客
郑渊洁 / 2000 / 学苑出版社
1.26 简单有趣，把童话写作现实。但我不喜欢他总把自己放进故事的做法。

Feb

被劫持的私生活
肉唐僧 / 2008 / 山西人民出版社
2.8-2.12 性解放比婚姻更重要

一九八四
[英国] 乔治·奥威尔 / 2009 / 上海译文出版社
2.22-2.26 封面之美收藏再合适不过。太有味道的一部思想史式的小说。

Mar

None.

Apr

泡与沫
[日本] 渡边淳一 / 1999 / 文化艺术出版社 天地图书有限公司
4.6 未完成版的失乐园，开放的结局让人印象很快模糊

小淘气尼古拉绝版故事
[法国] 勒内·戈西尼文 让-雅克·桑贝图 / 2007 / 中国少年儿童新闻出版总社 中国少年儿童出版社
老喽，满眼羡慕

狂热分子
[美国] 埃里克·霍弗 / 2011 / 广西师范大学出版社
4.22 趣味无穷的一本。狂热既是一种灵魂的疾病，却又是一种可以让社会和国家振衰起弊，死而复生的神奇工具。

被放逐的诗神
食指 / 2006 / 武汉出版社
再读一遍

May

None.

Jun

七十年代
北岛 李陀 主编 / 2009 / 生活·读书·新知三联书店
6.19 终于读完，断断续续反反复复，可以把这本书放下了。一群知识贵族，有的在抱怨有的在回味有的在感恩有的在怒骂，他们都选择坚持了，而从他们口中才找得到那些放弃了那个时代的一群背影。满眼的西欧和俄国名字几乎统统不认识，现在真的是最好的年代？

哈克和大鼻鼠全传
葛冰 / 2007 / 接力出版社

Jul

俗话说
东东枪 / 2010 / 新星出版社
7.15 一本靠谱的小黄书，充满机灵劲儿的短句。

异类
[美国] 马尔科姆·格拉德威尔 / 2009 / 中信出版社
7.18 时代+机遇+才智+勤奋+传承..其实有很多可以让人加速成长的元素需要把握。翻译不及译言。

Aug

写作
[法国] 玛格丽特·杜拉斯 / 2005 / 上海译文出版社
8.4 孤独以及锁住的自我。

在切瑟尔海滩上
[英国] 伊恩·麦克尤恩 / 2008 / 上海译文出版社
8.6 这样一个让人充满爱欲的循环，第一章难以控制的欲望和让人回味无比的结尾。

密涅瓦火柴盒
[意大利] 翁贝托·埃科 / 2009 / 上海译文出版社
8.12 来自意大利的公共知识分子，为人民的思考出谋划策

杜拉拉升职记
李可 / 2007 / 陕西师范大学出版社
8.26 断断续续地读完了，前一半明显感觉到作者有好多想教的东西，一股脑全都写尽，后半部就是外企出名的勾心斗角，并没有想象中的出彩，或许只能算是一个标签吧。工作中要不断思考这本书最大的真理。

Sep

枪炮、病菌与钢铁
[美国] 贾雷德·戴蒙德 / 2006 / 上海世纪出版集团
9.4 400页的猜想和解读，理性且有理有据，中国部分稍显模糊，这个民族过于奇妙。

忧伤的老板
左小祖咒 / 2010 / 上海人民出版社
9.7 一看就是老板没写太久的东西，料不够多，但加上一场新书发布会的话就变得蛮值了。真诚得动人。谢谢柴老板的慷慨相借。

过于喧嚣的孤独
赫拉巴尔 / 2010 / 中国青年出版社
9.16 别忽略我的生活，轰鸣也无法淹没我。

荒原狼
[德国] 赫尔曼·黑塞 / 2008 / 上海译文出版社
9.22 谢谢老矬，这本书就是我的图腾。周末去万圣书院买一本然后永远带在身上。在上班下班的公车上读完这一本电子版的图腾，灵魂无疑都是破碎的，在匆忙的脚步中追寻我的快乐的蛛丝马迹，内心也无疑是漂泊的，注定向往流浪注定向往无拘无束，却仍要在冰冷的都市里行走，对世界充满不解和怨埋，向远方和天际嘶声呼号，只为守住内心的芳华和脆弱，看似快乐却不堪一击，感受爱装作爱，告别平淡的时候对这世界便恩断义绝。

Oct

这个世界会好吗
[美国] 艾恺采访 梁漱溟口述 一耽学堂整理 / 2006 / 东方出版中心
10.15 半个月的早上排便时间读完的，趁着清晨时的清醒神经每天都得到不同的启发，虽然很多地方更像是历史人物评传，且当做还原历史了吧

野火集
龙应台 / 2010 / 生活·读书·新知三联书店
10.30 原来，大陆慢了至少20年。

Nov

人类曾经被毁灭
李卫东 / 1998 / 北京-九洲圖書出版社
11.4 很庆幸不是在高考之前看的，会很影响一个人的三观。个人最认同的是关于中介文明的阐述和赞许态度，文明从高端到低点、科学从真诚走向骗局也不过是两千年的事情，追根溯源便成了现代人需要做的事情。读的时候常会产生幻觉，似乎懂了某一个部分之后世界就会变成另一副样子，或者就得到了死后进入精神世界的通行证。谁都不能确定自己此时在做的究竟是不是正确的，期待对生命的透彻理解和对自然的幡然彻悟。跳出观念的裹挟，是否会更自由一些。

往事并不如烟
章诒和 / 2004 / 人民文学出版社
11.12 他们，拥有自己生活格调和节奏，让人艳羡，他们，对信仰的坚持，更显古风气韵，他们的离去，带走了一个时代。颇显古风的字里行间糅杂着恨和愤怒，以倾听者和亲历者的双重身份回忆的这段历史更显得弥足珍贵。对张伯驹、康同璧两人印象最深，那种隐忍的痛切痛心灵。

我们仨
杨绛 / 2003 / 生活·读书·新知三联书店
11.13 读完《往事并不如烟》之后读这本，同是回忆过往，一显一隐，一整一散，一恨一爱，而且这爱是如此偏执虚幻并隐去了很多真相，这刻意打造出的美好打动了如此多的人我没想到，不过看看便罢了，留下同情，爱未到如此深才更需去用心，为何不先问自己而艳羡别人，这种缺失无疑是明显的。

1912-1928-文武北洋
李洁 / 2006 / 广西师范大学出版社
11.28 “在这样的大环境里，人不可能不灰头土脸” 。最近很想知晓真实的北洋政治文化发展状态，那学者口中泛自由的感觉。本书虽未达到，但复活了北洋时期的一代名仕，越过遮掩，较真实地重现出来。为中国受到的误导，牺牲和浪费进行了根源性的剖析。震动最大为陈独秀的一篇。作者笔法很似南周体。

看上去很美
王朔 / 2006 / 人民文学出版社
11.28 他经常猜我是谁，来干什么。那时我也不知道我的使命是记录他，要是知道，我不会那么任性，会多留一些时间在他身边。一条庞大的生产线。孩子总会学"坏"的。想法那么多的小时候现在却都记不住多少了呢。调戏女生的技巧居然如此一致我很震惊。

Dec

世纪大审判
张世文 赵庆波主编 / 1997 / 内蒙古人民出版社
12.10 世纪审判 陈伟 "罪犯逃脱法网与政府的卑鄙非法行为相比，罪孽要小得多"

只有偏执狂才能生存
[美国] 安迪·格鲁夫 / 2002 / 中信出版社 辽宁教育出版社
12.11 敏感度，双向交流，转折点，10倍速因素，专注+独特

Catch As Catch Can
Joseph HELLER / 2003 / Simon & Schuster Ltd

流氓的盛宴
朱大可 / 2006 / 新星出版社
12.22 上升流氓的高度，失地者和失国者，他们往往失去而难于辨识拥有。在混乱中理清头绪，被时代淹没的语义，即使复活也一样丧失意义。虚华外表下暗流汹涌的中国，每个思考的灵魂都更难以解脱。
