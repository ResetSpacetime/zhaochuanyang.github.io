2020 阅读（下）

2020 阅读（上）

Jul

《中国人的气质》
[美国] 明恩溥 / 刘文飞、刘晓旸译 / 三联书店上海分店 / 2007
7.7 十分有利于深层次地理解和诠释我们自己

《译边草》
周克希 / 上海三联书店 / 2008
7.8 不知道怎么讲，技巧和细节倘说得多了，似乎它们总不必在这个层面上被讲出来。后半本反而觉得更好，分享了惊喜的感受。

《追寻记忆的痕迹》
[美国] 埃里克·坎德尔 / 喻柏雅译 / 后浪 / 中国友谊出版公司 / 2019
7.12 找到内核与问题。想象。人文背景与初始记忆。思维隐喻。鱼一生都在找寻水。保守的演化。触手、递质、环境、基因。反向书写（注释）。实验系统。群体情谊。

《赢家的诅咒》
[美国] 理查德·塞勒 / 高翠霜译 / 中信出版集团 / 2018
7.18 环境与短视，影响这么大，认知不够深。

《你永远都无法叫醒一个装睡的人》
周濂 / 中国人民大学出版社 / 2012
7.19 有些地方有启发

《大宇之形》
[美国] 丘成桐、史蒂夫·纳迪斯 / 翁秉仁、赵学信译 / 湖南科学技术出版社 / 2012
7.22 几何学被忽视了。地球太小了。

Aug

《把自己作为方法》
项飙、吴琦 / 上海文艺出版社 / 2020
8.1 挖下去

《心》
[日本] 夏目漱石 / 林少华译 / 青岛出版社 / 2016
8.4 那是一种怎样的执念呢，现在或许已不可解。便是有些非做不可的事，用尽一生都绕不过去。因为“我”一开始便被摧毁了，余下的便都是苟活。但倘不都全然拖出，就枉度了这一生。不如说是在等着什么。永远的逃离。“我脏了”与“我加害了”成了完整一体。永远也望不见的眼下/当下，孤独与隔阂，“死”的显名。

《我们赖以生存的隐喻》
[美国] 乔治·莱考夫、[美国] 马克·约翰逊 / 何文忠译 / 浙江大学出版社 / 2015
8.4 《理解》一章完美。摧毁定义。不如以行代言。回归个人隐喻。

倦怠社会
[德国] 韩炳哲 / 王一力译 / 中信出版集团 / 2019
8.5 停歇，延缓，禁止自我奴役

爱欲之死
[德国] 韩炳哲 / 宋娀译 / 中信出版集团 / 2019
8.8 爱欲的引诱力。对他者与边界的准许。想象力。矛盾与生命力。不安。

Sep

美的救赎
[德国] 韩炳哲 / 关玉红译 / 中信出版集团 / 2019
9.6 《美的救赎》 如此，每一个复杂的概念，便都需要去考证和挖掘它的历史了。以宏大施以刺痛。永恒与连绵的追忆，重塑完整，否定的权利。

一只好狼的故事
[波兰] 约瑟夫·魏尔康（文） / [德国] 彼得·尼克尔（图） / 刘海颖译 / 湖北美术出版社 / 2008
9.9 似乎有一条说不明白的隐线。或许就是这种误读出的阴冷，让我喜欢来自波兰的作品。

看不见的美国
[美国] 珍妮·拉斯卡斯 / 何雨珈译 / 上海译文出版社 / 2019
9.9 “没有必要”vs“很有必要”，这是意识层面上的截然不同。平滑的骗局，掩饰住所有的割裂和刺痛。这个后奥斯维辛时代的虚假道德。需要的新生活方式，似乎与“楚门的世界”里一样，细细观察并质问每一个人每一件物每一种服务，捅破边界，找到另一种自由。

扫地出门 : 美国城市的贫穷与暴利
[美国] 马修·德斯蒙德 / 胡䜣谆、郑焕升译 / 广西师范大学出版社 / 2018
9.20 人们被卷入了一个苦难的漩涡里，然后进入了“毫无办法”的境地，唯有外来者能理清这一切。我们有太多片面的整理与总结（书中的大量批注），但从未理出整体，窥以全貌，或是已丧失了这种能力？问题要被精确定义才能被解决吗？作者记录了足够多的面向，因而才有了拼凑的可能，能看出的，唯有：人已非人，无尽的轮回与永不止息的悲歌。

资本回报 穿越资本周期的投资
[英国] 爱德华·钱塞勒 / 陆猛译 / 中国金融出版社 / 2017
9.21 看了两本讲人的书，再看这本，一下子便上升了若干个层级。产业周期，无非是更大尺度下的经济样貌，就像看懂了山的呼吸，是需要更多信息与更多凝视的归因方法。但我并不觉得它是稳固的，马拉松公司必须保持长期持续的信息摄取，才能有更多改善和迭代的空间，这些基础和要素，国内统统不具备。但它确实给出了一种凝视的方法，足够的触手与信息归总，高频讨论和更新，可以促进准确度的提升。

乡下人的悲歌
[美国] J·D·万斯 / 刘晓同、庄逸抒译 / 江苏凤凰文艺出版社 / 2017
9.30 不得不说给我的启发很大，我们来自哪里，我们去向何方，我未知的过去如何造就我的人生，重大决策和心理预设。潜意识在被意识到时，更加痛和致命。

晴天有时哈哈哈
[日本] 矢玉四郎 / 彭懿译 / 二十一世纪出版社 / 2014
9.30 唯有报以哈哈哈

Oct

老实人
[法国] 伏尔泰 / 傅雷译 / 上海译文出版社 / 2017
10.8 这波荡起伏，仿佛浓缩了世界史

人与土地
阮义忠 / 九州出版社 / 2017
10.9 酒店翻过

Nov

图解道教
张梦逍 编著 / 陕西师范大学出版社 / 2010
11.8 初步了解下挺合适，很多没法细讲

节日万岁
[日本] 高木直子 / 香冰译 / 北京联合出版公司 / 2017
11.8 只对伸脖子那个有点兴趣……

马当家的饭桌
[韩国] 洪渊植 / 米糕译 / 后浪 / 湖南美术出版社 / 2020
11.9 我变成了母亲。

别再问我什么是嘻哈① : 人气作家×大学教授带你入门
[日本] 长谷川町藏、[日本] 大和田俊之 / 耳田译 / 拜德雅 / 上海社会科学院出版社 / 2020
11.28 清理偏见，拥抱变化之书。

知日·谁是增田宗昭
茶乌龙 主编 / 中信出版集团 / 2016
11.30 比那本茑屋书店的书也差太多了，内容不够深入，空间呈现也很不充分

晴天有时下猪
[日本] 矢玉四郎 / 彭懿译 / 二十一世纪出版社 / 2014
11.30 哇哦！鹅妹子嘤！

Dec

花森安治
[日本] 花森安治 / 王玥译 / 新星出版社 / 2018
12.1 那些我试图抹平的波动、起伏和不稳定，都是生活的搏动与周转

不方便，但很幸福
[韩国] 洪渊植 编绘 / 米糕译 / 后浪 / 湖南美术出版社 / 2020
12.4 表面的平淡，情绪起伏，虽然主诉于幸福，但也知道有更多苦难与不易。

犬中之猫
清华大学出版社 / 2020
12.17 太硬了，读起来可真累。不断想把赤狐解读为一种未成功的“人类”，文明之前的人类，或许也是一样的茹毛饮血吧。

海错图笔记
张辰亮 / 中信出版社 / 中国国家地理·图书 / 2016
12.23 若觉得吃捕有罪，就别写它们味道如何了。居高临下的视角和个人见解过于多，读着烦。没比照已然灭绝了的生物。
