Jan 5, 2014 | 阅读 | 本文共计 1.7k 字 | 阅读约需 6 分钟

Jan

《HUNTER X HUNTER》
富堅義博 yoshihiro togashi / 1998-2012 文化傳信有限公司
1.2

《20 世紀少年》、《21 世紀少年》
浦沢直樹 / 2008 / 東立
1.8 妈的结局还得看两本…上瘾啊。关于朋友、关于少年、关于绝望和希望。哎呀好喜欢健次、迦南。

《超智遊戲》1-18
甲斐谷忍 / 2009 / 天下出版(國際)有限公司
1.18

《寄生獸》

1.22

《JOJO 的奇妙冒險》 25
荒木飛呂彥 / 2004 / 東立出版社有限公司
1.27 生命本该如此。

《單身宿舍連環泡》 1-11
窪之內英策 / 1993 / 尖端
2.4 冲着搞笑看的，却是有味道的爱情故事出版后的第20个年头，原来真爱从未变过。

《小恐龙阿贡》 1-7
田中政志 / 2010 / 国际华文出版社

古谷實 / 2009 / 玉皇朝
2.11 挫男的超现实故事

《異色短篇集》 1-4
藤子.F.不二雄 / 大然
2.14

《破壞王》 1-8
刃森尊 村田秀雄 / 1993 / 東立
2.14 不习惯……看到8卷

《殘酷的童話》
由貴香織里 / 2005 / 東立出版社
2.15 华丽+暗黑，很美但很浅

《外天楼》
石黒正数 / 2011 / 講談社
2.16 推理范儿

《奇子》 1-3
手塚治虫 / 1981 / 講談社
2.19

《アナモルフォシスの冥獣》
駕籠真太郎 / 2010 / コアマガジン
2.19

2.20 直面心底

《Present for me 機器人的聖誕禮物》
石黑正數 / 2008 / 東立
2.22 “丢绳索的人反倒被绳索给套住，就是像这样。” ——《绳索超人》妈的爱死这篇了，看石黑正數的感觉有点像看麦克尤恩的短篇。

《萬祝》 1-6
望月峰太郎 / 2006 / 尖端
2.26

《守護她的 51 種方法》 1-5
古屋兔丸 / 2008 / 東立
2.27

《萬祝》 7-10
望月峰太郎 / 2009 / 尖端
2.28 读完这个又得一阵子看不进去青年漫了……

Mar

《悠游度过一天的24小时》
阿诺德·本涅特 / 2010 / 外研社
3.6 控制、反省、深入

《前卫之痒》
刘柠 / 2011 / 安徽教育出版社
3.26

《迈向另一个国度》
[日本] 森山大道 / 2012 / 广西师范大学出版社
3.28 执着于微小的喜悦和巨大的孤独中

Apr

《护生画集》
丰子恺 / 2005 / 上海人民出版社
4.4 涵芬楼读完。大智慧。极简。部分重复。第二、第四集中感触最多。

May

《哈佛最受欢迎的营销课》
扬米·穆恩 / 2012 / 中信出版社
5.1 跳出竞争；品类加法/乘法细化后消费快感边际递减；减法、分化、变形的创新；保持倾斜化

《自营销》
[美国] 亚历克斯·博古斯基 约翰·温莎 / 2012 / 浙江人民出版社
5.3 过于广告人的角度，案例浅薄

《助推》
[美国] 理查德·H·泰勒 卡斯·H·桑斯坦 / 2009 / 中信出版社
5.8 锚定法则、易得性法则、代表性法则。刺激反应相容性效应。冷热移情差异。1/n论断。默认选项、理解权衡、反馈、动机、预计错误、结构性符合选择。

《因自由而美丽》
黎戈 / 2013 / 新星出版社
5.11 平和、淡然的女子

《進擊的巨人》
諫山創 / 2011 / 東立出版社
5.14 补完45话mark

《精益创业》
[美国] 埃里克·莱斯 / 2012 / 中信出版社
5.14 开发→测量→认知；目标用户→假设→数据指标→产品功能；基准→优化至理想状态→坚持or转型；看板管理；价值定义“向顾客提供利益”；工作根据“经证实的认知”数量来衡量；市场验证后才引入营销。

惡之華
押見修造 / 2011 / 東立出版社
1 5.14 摔！仲村明明是个萌妹子么！
2 5.16 神展开。想想觉得我更像是佐伯那种隐藏着“恶”的人吧。

暗黑对决 1
壬生吕瓶 桥本以藏 / 2010 / 台湾东立出版社
5.21

惡之華 3-7
押見修造 / 2013 / 東立出版社
5.21

5.24 这又臭又长的后半段……

《漂流網咖》 1-7

6.14

《ネムルバカ》

Jul

《MBA 教不了的创富课》
老雕 / 2011 / 当代中国出版社
7.2 知识只有理解到这种信手拈来的层面才真算是有用吧。杠杆率、价值链和竞争链的阐述受益匪浅。

《写给大家看的设计书》
Robin Williams / 2009 / 人民邮电出版社
7.6 字体部分对中文排版来说真是没什么参考价值……前半经典

《乡土中国》
费孝通 / 2006 / 上海人民出版社
7.22 礼俗社会/法理社会、差序格局和西方文化模式的阐释醍醐灌顶，喜欢在大城市呆着多数就是因为这个原因吧。

《给我的孩子讲死亡》
[法国] 爱玛努埃尔·于斯曼·佩兰 / 2011 / 重庆大学出版社 / 楚尘文化

《史玉柱自述》

《给我的孩子讲骑士与骑士制度》
[法国] 阿兰·德缪杰 / 2013 / 重庆大学出版社 / 楚尘文化
