【逢十】推荐电影第三期

2015-08-23 | 阅读 | 本文共计 1.7k 字 | 阅读约需 5 分钟
Comment

【逢十】系列第三篇。2015年的第21-30部观影小结，从1月29日至2月13日，追评时间7月19日。

我的个人建议分为四档：建议不看、爱看不看、推荐观看、不得不看。

相关豆列：【2015观影】

1.小小迪特想要飞（豆瓣8.3 / IMDB8.1）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 死神不想要他

  • 现在追评：

    • 在看完《重见天日》之后看这一部，当事人亲自复述，鲜明的赫尔佐格式的纪录片。赫尔佐格在访谈里有谈到，当事人的故事是偶然听到的，然后就决定要拍一部纪录片，但当事人的心理压力巨大，很久之后才答应。于是有了这部极其精彩的个人口述史。
    • 一切起于他的梦想，却几乎被摧毁，最后求生与对未来的坚持都成为了他的本能，无所畏惧。导演好奇心旺盛，不容漏下任何一个细节，事无巨细地全部记录了下来。当然，这已是赫尔佐格电影最有特色的部分。
    • 另外，电影最后的航拍片段，实在不容错过。

2.夜莺（豆瓣6.4 / IMDB7.0）

  • 个人建议：

    • 建议不看

  • 当时短评：

    • 想念阳朔了……

  • 现在追评：

    • 看影评的时候才发现是《蝴蝶》的中国版（蝴蝶我还没看过），看完对蝴蝶也没兴趣了。大BUG无数，小演员演技差到极点。

3.狐狸猎手（豆瓣7.3 / IMDB7.1）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 玩物

  • 现在追评：

    • 不如去看《大象》；不喜欢的点在于：虽然也是真实事件改编，但是加了过多的“动机”，突然的转折才是这类故事的重点。
    • 比《点球成金》沉重了很多，梦想与励志的背后，权力和欲望的操纵；一代人梦想的背后，是一个空旷的庄园，几个拼搏的人，以及一个掌权者。
    • 不过，杜邦童年和心理阴影的形成刻画得确实不错，喜欢研究心理学的人一定会喜欢。

4.未来学大会（豆瓣7.5 / IMDB6.5）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 硬科幻。取消生命存在的物质基础和时间单位的存在形式，一定程度上等于了突破生命的限制，但同时也被囚禁于过去。这片子的着眼点则是两个现实的纠葛，没有对物质进行讨论，不够过瘾。

  • 现在追评：

    • 是根据波兰科幻、哲学作家斯坦尼斯拉夫·莱姆的短篇改编的。由动画部分和真人部分组成。这是一份来自过去的对虚拟现实的想象，真的不必要求过高。
    • 去年读过一本《第三次浪潮》，作者对于现在的社会和科学的发展有不可思议的预见性，具体的研究方法不得而知，但一定是一种严谨的方法论，脑洞也是基于事实的。
    • 莱姆有几部作品都有探索永恒的味道，脱离肉体的智慧的存在，反而这样的故事更能引起对人类死神的反思。

5.岳父岳母真难当（豆瓣8.0 / IMDB7.0）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 不好笑= =……

  • 现在追评：

    • 别人安利的，很不满意，云里雾里不知所云，一个笑点都get不到，对法国喜剧有点太放松警惕了……

6.爱很怪（豆瓣7.8 / IMDB6.7）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 好喜欢。两个人唱歌、亲吻、拥抱、回忆，处处充满生活的印记。多少思绪，最终都只是一句“我好想你”。记忆越是简单，追忆时越是如排山倒海。

  • 现在追评：

    • 豆瓣友邻里有提到这部电影与《东京物语》和《明日之歌》相似，我没看过，暂不做对比。
    • 除了题材之外，城市、情感、生活，甚至配乐（肖邦的钢琴曲），都是有机的整体，生活随着时光静静流淌。他根本不像一部同志题材的电影，而是简简单单地在陈述一段凝滞在岁月中的情感片段，一段封存在某个人内心里的记忆。

7.南极料理人（豆瓣8.1 / IMDB7.2）

  • 个人建议：

    • 不得不看

  • 当时短评：

    • 吃炸鸡痛哭那段真是满满的吉田修一味儿啊，只是这部人物的背景平淡了一点。这南极像个中年大叔的乌托邦童话。

  • 现在追评：

    • 短评可能涉嫌剧透，但是无论怎样，看到吃炸鸡那一段也会动容（类似的感觉最经典的要数《生活多美好》，也是不得不看），我无聊看几遍都会动容。
    • 日本电影里总是有这样奇怪的点，经过长长的铺垫，最后仅仅通过一个不经意的细节（可能是句台词，或者一个动作），让情感冲开闸门倾斜而出，起先惊诧而后悲喜交加，任由自己被这翻涌的情绪淹没。
    • 另：导演冲田修一和作家吉田修一都是这样细节的收集者，上一次领教是《横道世之介》

8.盗钥匙的方法（豆瓣8.4 / IMDB7.4）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 快过年了，看点儿喜剧

  • 现在追评：

    • 香川照之在《洋芋片》里也有出演，总觉得他眼熟，结果发现他在《鬼子来了》里面演那位花屋小三郎……
    • 纵使互换了人生，习惯还是能引导你找回自己。电影在这点上正能量了一下。

9.圣人文森特（豆瓣7.6 / IMDB7.3）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • bill murray控、melissa mccarthy控、发音控。这类型看得少。笑了哭了。过年了。

  • 现在追评：

    • 没办法，就是喜欢 bill murray 这个家伙，他演了太多韦斯·安德森的电影，他们的感觉实在搭得发指。
    • 没办法，太喜欢 melissa mccarthy ，《伴娘》里的形象无法磨灭。
    • 剧情上还是有硬伤，早些把世界的残酷展示给孩子，到底是好事还是坏事，这事儿一直值得讨论。

10.IT狂人说明书（豆瓣9.5 / IMDB-）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • TAT，终于还是终结

  • 现在追评：

    • 纯个人喜好。当然，看这集的前提是你补完了这部名为《IT狂人》的经典英剧。
