Jan

《爱的地下教育》
彭浩翔 / 2010 / 甘肃人民美术出版社
1.1 刁钻古怪。生来犯贱和相见恨晚症候群很精辟。

《佛祖在一号线》
李海鹏 / 2010 / 文化艺术出版社
1.23 聪明人，别做蠢事

Feb

None.

Mar

None.

Apr

《遇见未知的自己》
张德芬 / 2008 / 华夏出版社
怜惜时间的话，别看

《金蔷薇》
[俄国] 康·帕乌斯托夫斯基 / 2007 / 上海译文出版社
4.18 受到的启发无尽无休

《美国讲稿》
[意大利] 伊塔洛·卡尔维诺 / 2008 / 译林出版社
4.19 些许繁复，但就理论和思考之于文学重要性的探讨很深刻，文学不应仅仅是记录和反映，同时也是总结。

《呓语道破》
杨志强 / 2010 / 生活·读书·新知三联书店
4.22 是作者的总结陈词，更是对读者的抛砖引玉，获得也即失去，谁又曾真正执着于真相。

《春天责备》
周云蓬 / 2010 / 上海文艺出版社
4.24 好像蛮久之前把随笔都读过了，最近两天才把诗歌都通读了，在下不才，对后现代或前卫之流基本无感，只喜欢其中五六首诗。随笔嘛，印象中还是很磨叨..有一股沈阳人的懒劲儿和空味儿。总之丰富的积淀和多年对弱势群体的关注让老周有很不同的气场，而这些梦呓般的诗句还是搭配着旋律才更显动人。

《最后的罂粟》
祝勇著 冰冷川绘 / 2004 / 三联书店
4.24 十幅梵高的肖像都很赞，但旁边的文字却恰是十篇不太出彩的文字。祝勇，不认识，但文字内敛含蓄，想言说的话都像糖果含在一层又一层的包装中一样，藏而不易得，引人反复斟酌。还有，我对冷冰川刻画的女人体实在不感兴趣，流连山水自然之味，尽管已被抛却了色彩。

《我的精神家园》
王小波 / 2006 / 中国人民大学出版社
4.28 幽默、犀利，读如此随性的杂文常常不禁莞尔，又收益颇丰。性格和视角无不影响着人对事对物的观感，捍卫选择的正确和重要，自信责任与承诺，喟嗟不若善己。

《世说新篇》
黄苗子 / 2006 / 生活·读书·新知三联书店
4.30 至少这些牛逼闪闪的人，真的存在过。

May

《动物农场》
[英国] 乔治·奥威尔 / 2007 / 上海译文出版社
5.18

《九栋》
邹静之 / 2010 / 法律出版社
5.22 奶奶，春天，懵懂，死亡，赌博，病退，厕所，偏方，药狗，这九个故事看得我心悸。作者带着点诗性的描写给读者留着想象的余地，感情充沛。一个时代的侧写读来总是别有滋味，最愿意聆听这样的故事，它就发生在这片土地之上，却距离我们那般遥远。看得出作者的真诚，和一腔被克制的锋芒，他的坚持，才有了整本书中文字平静的流淌。

Jun

《雪泥爪印》
黄苗子 / 2006 / 生活·读书·新知三联书店
6.2 黄苗子周游列国，中日片段更为出彩

《卯个人》
王卯卯 / 2010 / 北方妇女儿童出版社
6.11 这次绘本厚了点儿，所以跟老婆看了近一小时。卯卯有这个人有一种动人的真实感，有她独特的观点和方式，努力地认真地生活。绘本读来甚为真切，触动也都结实而有力，那不只是兔斯基的哲学，更是对曾经的完美总结和和反思。恩，还照着很诡异的说明书把万年历装好了...

《那些有伤的年轻人》
庄雅婷 / 2010 / 华文出版社
6.14 不知晓庄雅婷是何许人也，也没有兴趣，此类文字着实不适合两天内读完，会越读越腻，前一百页读来热闹后一百页渐次无感，第四部分太适合女性多读读。想来不过也都是些常识，只是人们不敢点破，总要借助第三方的力量。想得愈多伤得愈深，何不“一切如流水掠过，湿过身就好”。

《樱桃之远》
张悦然 / 2004 / 春风文艺出版社

《十爱》
张悦然 / 2004 / 作家出版社

《葵花走失在 1890》
张悦然 / 2003 / 作家出版社

《这些都是你给我的爱》
安东尼 / 2010 / 长江文艺出版社
6.20 23分钟读完 一个人瞎想&YY到这个境界也蛮极品了 你笃信什么 确定一种感觉然后信口胡诌出的文字就能把你治愈了吗 你热衷于把这种书放在床头 那么你依旧是那个不会思考不会自己寻找快乐享受生活的人 这是不会改变的

《毛姆读书随笔》
毛姆 / 1999 / 上海三联书店
6.20 “没有一本一劳永逸的书。没有终极智慧在什么地方等着你。美好之人生，不外乎各人顺其性情，做好份内之事”。终于买来读了，毛姆渊博的学识和平易的口吻真的让人痴迷。通读这一番指点后，开始对好多大部头的书有了兴致。闲来读书是兴趣之事，不能牵强更不应随意，究人生奥义探人生谜题，切记认真体悟。读到的，都是诚恳之言经验之谈，满书闪闪发光的观点让人艳羡。思考才是人生之大快。

Jul

《当我们谈论爱情时我们在谈论什么》
[美国] 雷蒙德·卡佛 / 2010 / 译林出版社
7.30 有意识地抽取，不刻意补充细节，开放结尾，与其说极简不如说是随意，突破形式，大块的留白，隐形的情节。一次一两篇蛮好，一次读多了恐会无味。

Aug

《青灯》
北岛 / 2008 / 江苏文艺出版社
8.4 喜欢第二辑，第一辑很有看九栋时的感觉...“我们自以为与时俱进，其实在不断后退，一直退到我们出发的地方”

《一点法国》
[法国] 让·雅克·桑贝 / 2010 / 中信出版社

《刀锋》
[英国] 毛姆 / 2007 / 上海译文出版社
8.11 不急躁，对人随和，慈悲为怀，丢掉一个我字，不近女色，无所谓代价。毛姆带你看世界，享受精神生活。

《大家都有病》
朱德庸 / 2011 / 现代出版社
8.12 某华润超市 某些碎语蛮好，总体不及前言。
