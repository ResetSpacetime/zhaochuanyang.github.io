Jan

惶然录
[葡萄牙] 费尔南多·佩索阿 / 2012 / 上海文艺出版社
1.2 无感

《怪物特別篇：沒有名字的怪物》
Emil Sebe, Jakub Faroubek, Klaus Poppe等 監修：浦沢直樹 / 2009

《赫尔佐格谈赫尔佐格》
[英国] 保罗·克罗宁 / 2008 / 文汇出版社
1.14 向神秘致敬，导演也是一种编辑工作，把答案和问题都放在作品里，和而不同，徒步认知世界的广大，四海为家，想象力超越空间

美国种族简史
[美国] 托马斯·索威尔 / 2011 / 中信出版社
1.25 可见往往不是种族问题，而是文化问题、态度和观念问题、适应和交往问题。对于复兴，更重要的恰是人文资本，和元朝后中国开始没落的观点也能连接。

《中华散文珍藏本》
汪曾祺 / 1998 / 人民文学出版社
1.28 满是用心记下的美

《拉维尼娅的学校》
[意大利] 比安卡·皮佐尔诺著 [英国] 昆廷·布莱克绘 / 2013 / 明天出版社
1.30

《三闲集》
鲁迅 / 1980 / 人民文学出版社
1.30 寂静浓到如酒，令人微醺

《好小子-童年故事》
[英国] 罗尔德·达尔著 [英国] 昆廷·布莱克绘 / 2009 / 明天出版社
1.30 “生活是严峻的，你最好快点学会对付它。”

《人景·鬼话》
杨炼 友友 / 1994 / 中央编译出版社
1.31 意大利！第三人称的漂泊。附录的对话好棒~

Feb

《恐怖分子》
约翰·厄普代克 / 2009 / 人民文学出版社
2.4 熔炉、零落、工具。“信仰”们的暗斗被描绘得特别可笑。

《介绍我自己的思想》
胡适
2.7 拆散和重组的工作

《怎样读书》
胡适 / 1936 / 上海一心书店
2.9 读书的升阶，各人路径不同。所见皆是书，大赞同。

《九章算术》
曾海龙 / 2006 / 重庆大学
2.11 kindle版，略读。问题集，只见答案，能一窥生活。数学用语古早已有之。“盈不足”开始经济，方程部分有点晕菜。对没有符号时的数学思维方式好奇。

《独闯天下》
[英国] 罗尔德·达尔著 [英国] 昆廷·布莱克绘 / 2009年03月 / 明天出版社
2.12 罗尔德的笔下残酷也变成好奇

好心眼儿巨人
[英国] 罗尔德·达尔著 [英国] 昆廷·布莱克绘 / 2009 / 明天出版社
2.15 啊！太喜欢了！

中国小说史略
鲁迅 / 1973年 / 人民文学出版社
2.19 千年不变之中国

Mar

《意大利古建筑散记》
陈志华 / 1996 / 中国建筑工业出版社
3.17 收了实体书来做标记。读了两遍仍有玩味之处，想把它做成配图版。

《万物》
[德国] 延斯·哈德 / 2015 / 后浪 / 北京联合出版公司
3.28 一本满是真相的书，隐喻多得像一首诗。提醒你生命是一种现象、人是一种细菌、进化从来没有停止、生死是几十亿年前的一次选择……

Jun

《洛书河图》
阿城 / 2014 / 中华书局
6.4 阿城更像个翻译家而非学者，这未尝不是一条线索。个人觉醒。洛书河图读完。环境造人，始于礼失，礼失而攻击不绝。孔子老子的觉醒后，陷入千年迷思和迷失。天象变化和大陆架变化的理据让山海经变得真实。任何人造事物都会异化甚至消失。无对错，伪真相。

《谷物大脑》
[美国] 戴维·珀尔玛特 克里斯廷·洛伯格 / 2015 / 机械工业出版社
6.6 奇妙，严格控制碳水化合物摄入的观点第一次接触。吃油方面有体会，初榨橄榄油真是健康又美味。DHA、白藜芦醇、姜黄、维D、益生菌、椰子油、α-硫辛酸。

《从0到1》
彼得·蒂尔 布莱克·马斯特斯 / 2015 / 中信出版股份有限公司
6.7 敢于冒险、敢于承担风险的，才是企业家。秘密与垄断。批判渐进主义、风险规避、自满和扁平化的部分比较新奇。工程、时机、垄断、人员、销售、持久、秘密。

《达维多维奇之墓》
[南斯拉夫] 丹尼洛·契斯 / 2014 / 中信出版社
6.9 博弈：“存在的生与死”与“不存在的生与死”。轮回中的自我意识。精神上的日期序列。

《树上的男爵》
[意大利] 卡尔维诺 / 2012 / 译林出版社
6.11 疏离人，然后成为人。因为准则。

《克罗诺皮奥与法玛的故事》
[阿根廷] 胡里奥·科塔萨尔 / 2012 / 南京大学出版社
6.11 聪明的坏蛋。中介物。散文诗。非专业解构的世界。

《鱷魚街》
布魯諾·舒茲 / 2012 / 聯合文學
6.14 破碎之花。破败人生。色彩爆裂、黑色漩涡。飞鸟、深渊。罂粟。细线。无尽头。抛弃时间。父亲。灵感。魔法。飞升、坠落。失望。伤口撒盐。清扫。

《倒数第二个真相》
菲利普·迪克 / 2015 / 四川科学技术出版社
6.19 写得太颠儿了，逻辑有硬伤。但恐慌的感觉很到位。

《死亡迷局》
[美国] 菲利普·迪克 / 2015 / 四川科学技术出版社
6.20 有一点点粗糙，但是读来顺畅。把各种宗教的优势合而为一，并具象化。外形破坏者有千面之神的感觉。唯一为慰藉，是死亡。

《城记》
王军 / 2003 / 生活·读书·新知三联书店
6.22 定位不仅大误还多变，对科学无知无意识。北京无ZF，人民如蝼蚁，喉舌改造同步配合。见识短，愚蠢的循环。权力与个人绑定，决策和资源垄断。当下争议之事不如搁置。

《沒有女人的男人們》
村上春樹 / 2014 / 時報文化
6.27 意象太少。印象深刻的唯有八目鳗，《木野》那篇。

《不被理想束缚的生活》
[日本] 金子由纪子 / 2015 / 北京联合出版公司·后浪出版公司
6.27 直视内心。满满的正确观念，一本好鸡汤。想起那句“我想成为坐在路边鼓掌的人”

《幻之光》
[日本] 宫本辉 / 2013 / 上海文艺出版社
6.29 死亡背面。找他的魂魄。

Jul

《诗人的生活》
E.L.多克托罗 / 2015 / 上海文艺出版社
7.5 敏感而充满意象的生活，喜欢自问、独处，唯恐疯狂的好奇心落回深谷。细节太多、情绪转换太快读起来略吃力。

《灿烂千阳》
[美国] 卡勒德·胡赛尼 / 2007 / 上海人民出版社
7.13 畅销书罢了

《商业百科》
英国DK出版社 / 2015 / 电子工业出版社
7.14 总结了很多商业思想的提出和演化，最赞的在于把很多商业畅销书连成了线，带着逻辑同时思索几本书的思想，能得到一个不错的框架。秘密在于，从0到1，从99到100，从100到101，以及从999到1000。

《阿兰的战争》
[法国] 埃曼努埃尔·吉贝尔 / 2015 / 北京联合出版公司 / 后浪
7.25 另类的，一个没有打过仗的军人的战争史。被艺术、生活、友人改变了人生轨迹。广度够了，深度没到的人的忏悔录。“所有我爱过的人，几乎后来都再次见到了”是件幸福无比的事。

《增长黑客》

《走进老房子》
汪森强 卢庭芳 / 2005 / 江苏美术
11.15 购于宏村树人堂，当时被满眼的木雕搞得晕头转向，这本书帮了大忙。布局、木雕、题额、楹联，文化与生活相依存，只可惜其中之人多已不再，必然也佚失不少。家家户户惟愿“一生平静”。

《回族清真美食文化》
洪梅香 刘伟 / 2010 / 宁夏人民出版社

《月光落在左手上》
余秀华 / 2015 / 广西师范大学出版社
11.16 字里行间的满是疼痛的伤疤，然而因为爱而无法斩断今生，甘愿如一棵狗尾巴草，卑微地活着，等待把自己交还给大地。月光，无言，光明的黑暗，雪之恶。

《我们的家·我们的房子·我们的农庄》
[瑞典] 卡尔·拉松 / 2012 / 新星出版社
11.20 难忘那一缕阳光和身后的花园

《童年与故乡》
[挪威] 古尔布兰生 / 2010 / 新星出版社
11.22 吃栗子的时候看，不禁觉得有类似的地方：一会儿吃到了甜的；一会儿吃到了怪味的；一会儿吃到了黑掉的。

《局外人》
[法国] 加缪 / 2015 / 上海文艺出版社
11.23 没有人去设想那个“如果”，它就不存在吗？

《死水恶波》
[美国] 蒂姆·高特罗 / 2013 / 人民文学出版社
11.26 《空路不堪望》和《沟中小蛙》，有点像在看冰血暴，微微用力过度的还原，格外适合写人生的荒诞，于不合理处解读合理

《大鱼》
[美国] 丹尼尔·华莱士 / 2013 / 北方妇女儿童出版社
11.27 父亲、英雄、大鱼。信念、意志、梦想。

Dec

《身边的江湖》
野夫 / 2013 / 广东人民出版社
12.24 前面对自由洒脱、江湖气焰的刻画，在后几篇对时代戾气的吐槽下显得格外出挑，悲歌和史诗聚合在人的身上，成了时代的活化石，一一找到去路，归隐、归来或皈依的时候，默许了时间的创伤，静候黎明
