备忘栏

  • ed2k 下载
    • verycd 后增加 “.gdajie”，即 www.verycd.gdajie.com/XXXX，跳转网站获取链接，逛电驴 verycd.gdajie.com搜索，simplecd.me搜索
  • bilibili 下载
    • bilibili.com，bilibili 后面加 "jj"
  • Potplayer 快捷键
    • Enter：全屏
    • Ctrl + G：打开连续截屏工具
    • Ctrl + E：截存当前图像
  • Win 操作
    • 输入法
      • Ctrl + Shift + F：切换简体、繁体输入
    • 截图
      • Win + W，在弹出的 Screen Sketch 中编辑
      • Win + Shift + S，截取屏幕部分内容，储存至剪贴板
      • Win + PrtScn，截全屏，自动存至系统默认 Picture - Screenshots
      • 电源键 + 音量键"+"，自动存至系统默认 Picture - Screenshots
    • 关闭触摸屏
      • 按 Win 键，搜 “Device Manager / 设备管理器”，并进入
      • 找到 “Human Interface Devices / 人机接口设备”，下拉框打开
      • 找到 “HID-compliant touch screen / 符合 HID 标准的触摸屏”，右键 “禁用”（有两个，一般是第一个）
    • 快捷键
      • 新建虚拟桌面：Win + Ctrl + D
      • 关闭当前虚拟桌面：Win + Ctrl + F4
      • 最小化所有窗口/恢复：Win + M / Win + Shift + M
      • 临时查看桌面：Win + ,
      • 快速搜索：Win + Q
      • 屏幕显示快捷键：Win + P
      • Ink 快捷菜单：Win + W
      • 创建新 OneNote 笔记：Win + N
      • 放大/缩小：Win + (+ / -)
  • 第三方邮件客户端绑定
    • 腾讯企业邮
      • 接收
        • 服务器：pop.exmail.qq.com
        • 端口：995
        • 安全类型：SSL/TLS
      • 发送/外发
        • 服务器：smtp.exmail.qq.com
        • 端口：465
        • 安全类型：SSL/TLS
    • 网易企业邮
      • 接收
        • 服务器：pop.ym.163.com
        • 端口：995
        • 安全类型：SSL/TLS
      • 发送
        • 服务器：smtp.ym.163.com
        • 端口：465
        • 安全类型：SSL/TLS
  • Mac 操作
    • 截图，储存至剪贴板（而非桌面）
      • command + control + shift + 4，划取屏幕部分截屏，而不储存至桌面
    • 放大、缩小：command + "+" / "-"
  • 终端操作
    • cat + 文件名 显示，vi + 文件名 编辑，ssh-copy-Id 直接复制
  • 访问特定文件夹
    • finder 下 cmd + shift + G，前往特定文件夹，如 isa
  • atom 快捷键
    • cmd + t 模糊搜索目录下文件 ctrl + 0 焦点移至左侧目录 a / d / m 增删改
  • SSR 配置
    • 打开 .exe 文件；打开 txt 文件，全选复制；SS - 从剪贴板复制地址；右键 - 系统代理模式 - PAC；服务器 - HX 号 - 随便选一个服务器
  • 故宫博物院数字展品
    • 提示“插件下载说明”时，复制关键部分链接 "imagepath=" 之后，"&mis" 之前，添加到 "http://image.dpm.org.cn/dyx.html?path=/tilegenerator/dest/files/image/" 后面，即可打开（部分展品不能查看）
  • tree 结构归档
    • "brew install tree"，"tree" 显示所有文件树形图，"tree -L 2 -N" 显示两级树形图（并正常显示中文），"tree -L 2 -N > README.md" 直接将结果输出至文件中（默认在当前目录下）
  • Hexo 避免 HTML 源文件被渲染：npm install hexo-processor-copyassets --save
