2023 阅读

2023-04-10 | 阅读 | 本文共计 2.2k 字 | 阅读约需 7 分钟
Comment

Jan

《间谍》
[巴西] 保罗·柯艾略著 / 孙山译 / 北京十月文艺出版社 / 2018
1.10 不太行

《走进魔法店》
詹姆斯·R·道迪著 / 皛琳译 / 天地出版社 / 2019
1.14 播种要抵达心里。虔诚与意志。相信的力量。

《失落的爱》
[葡萄牙] 卡斯特洛·布兰科著 / 王全礼译 / 中国对外翻译出版公司 / 1993
1.16 生生拆成异地恋…爱比生命重要。心乃生命力之源泉。缝缝补补的心经不起大浪。

《与狼共度》
法利·莫厄特著 / 广西师范大学出版社 / 2022
1.21 动物和人相比，谁更懂尊严？狼和人相比，谁更喜爱诽谤和杀戮？什么是稀缺，什么是富足，什么是贪婪？甚至描述了友情、爱情、族群情谊。生命们在需要保持共存时，必要的代价又应是如何？你死我活显然不是唯一答案。

《地。關於地球的運動》
魚豊 / SCALY / 尖端出版社 / 2022
1.21 观察天道啊。自信和勇气的加持，无人不是在突破桎梏，死亡亦不可怕。一次次失败，一次次蒙尘，但一旦传播，便摧枯拉朽。这也是一种垄断（解释权），脱离了地面，以天来指挥地（也可能只是在以现代观点重释过去），那便可作为一种隐喻来理解了。天与地的平衡，又该如何达到呢。

《鲸之殇》
法利·莫厄特著 / 高建国、李云涛译 / 广西师范大学出版社 / 2021
1.23 人，或许已不能称之为人。过多的杠杆，过多的盈余，造就了太多恶魔。

《金之國與水之國》
岩本奈緒 / 平川遊佐 / 尖端出版 / 2020
1.26 分镜很美

Feb

《未踏之地》ランド 1-6
山下和美 / 講談社 / 2015
2.9 开头真的不太看懂…后来…似乎用空间和文字做了隔离，两种社会平行着。文字在“此世”中起到的作用，似乎可以理解为凝聚力量，构建叙事，加速变革的发生，可见文字这种第三方工具（技术）在整合上的巨大优势。山里那些弃儿比较有趣，他们的存在是为了颠覆。

《最重要的事，只有一件》
[美国] 加里·凯勒、杰伊·帕帕森 / 张宝文译 / 中信出版社 / 2015
2.12 目标与实际做的，要完成匹配。观点没毛病，但需要很多方法指引。人类对“大”“赢”都有太多执念了，不如仅仅放在小而美的范畴下，做好一切长周期的应做之事。

《不工作的世界：AI時代戰勝失業與不平等的新經濟解方》
丹尼爾．薩斯金 Daniel Susskind / 周玉文 / 天下文化 / 2020
2.14 想重新驾驭闲暇？那就重新造一个世界吧，还要再“复活”一些人。

《敦煌：纪念敦煌藏经洞发现一百周年》
敦煌研究院编著 / 朝华出版社 / 2021
2.16 真美啊，都是严谨的故事与传说。时代孤品。几乎是另一套语言体系，依附在大量、充足的环境信息中。感叹逝去的土壤与时代感，追不回来。

《小巨人：不做大也能成功的经营新境界》
[美国] 保·伯林翰著 / 王珺译 / 中信出版社 / 2013
2.17 是对很多大公司组织形式和管理模式的质疑。反而这些公司，都有从土地里生长出来的感觉。生态位，只扎根，不越界。社会型企业，更难，但也更难被替代。

《巨婴国》
武志红著 / 浙江人民出版社 / 2016
2.18 真特么辣眼睛

《四葉妹妹》 1-6
东清彦著 / 黃炳雄译 / 台灣國際角川書店 / 2004
2.18 哈哈真的很可爱啊

《The Art of Impossible : A Peak Performance Primer / 跨越不可能》
[美国] Steven Kotler / Harper Wave / 2021
2.19 确实在从实质意义上谈论身体、谈论由内而外和身体的表现，废除旧观念，很多执行建议，与最近读的几本书都有联动。尤其要注意复利的积累。

Mar

《白银资本：重视经济全球化中的东方》
[德国] 贡德·弗兰克著 / 刘北成译 / 四川人民出版社 / 2017
3.11 可以算作一种重新审视，或许现在看来是边缘与中心的关系，但早已无任何中心可言。资本的流动，投机的计算。无形的货币，还如何控制呢。小冰河以很隐秘的方式影响着朝代历史，很有意思。

《重金属音乐》
[比利时] 雅克·德·皮埃尔庞编、[法国] 埃尔韦·布里绘 / 张萌萌译 / 四川文艺出版社 / 2023
3.11 不说内容了，至少译名都很用心。发现我不过是赶上了金属音乐时代的尾巴，但这真的是好东西。

《鲨鱼》
[法国] 贝尔纳·塞雷编、[法国] 朱利安·索莱绘 / 陈潇译 / 中国友谊出版公司 / 2023
3.12 我们对世间万物的了解，都是如此贫瘠…

《卢利尤伯伯》
[日本] 伊势英子文&图 / 赵玉皎译 / 二十一世纪出版社 / 2009
3.12 再也没有坏过

《在西伯利亚森林中》
[法国] 西尔万·泰松 / 周佩琼译 / 上海文艺出版社 / 2015
3.12 如果作者的心境是赴死，还情有可原，然而并非如此。连内观都算不上。

《清算：华尔街的日常生活》
[美国] 何柔宛著 / 翟宇航、宋岳、张樹沁、胡凤潮译 / 华东师范大学出版社 / 2018
3.20 对投行标榜的精英感、傲慢感、危机感、不安全感，还有专业演员式的装逼范儿都有了更深一层的了解。作为逐利者的市场，谁先发现利，谁先搞定利，效率和结果为唯一导向，利益分配是唯一核心问题。一整套空虚的符号体系，抽象且简单，根本无法应对风险，即便有，也与任何人无关。赢家赢在控制，而非所谓认知、洞察。

《竞争优势：透视企业护城河》
[美国] 布鲁斯·格林沃尔德、贾德·卡恩著 / 林安霁、樊帅译 / 机械工业出版社 / 2021
3.24 信息量真大。小规模需要可控，大规模需要谨慎。战略咨询对于大企业，确实是重要的。“将高质量信息与低质量信息相结合并不会得到平均质量的信息，最终所得仍是低质量信息。”时刻保持对需求的敏感。在核心市场，早早赚足超额利润，保持多项优势，品牌才能真的形成（成为一种生活叙事）。

Apr

《熊在吗》
沈志军、朱赢椿著 / 浦睿文化·湖南文艺出版社 / 2022
4.1 “熊变成了光！”

《妈妈走后》
[美国] 米歇尔·佐纳 / 蔡雯婷 / 北京联合出版公司 / 2023
4.6 “你并无多少惊世狂才”、“你知道我意识到什么了吗？我从来没有遇到过像你这样的人。”、“她以无人能比的热忱观察我，并将永不枯竭的爱意投向我。”、“如同站在风暴之眼，平静地看着周围的一切分崩离析。”母女情，似乎透过本书窥见了一些，深层的怜惜，隐秘的保护，真情的切实存在，太多无计较的付出。这些，再加多一层文化隔阂，放大出了更多的共情。

《深时之旅》
[英国] 罗伯特·麦克法伦著 / 王如菲译 / 文汇出版社 / 2021
4.9 对原住民和传说的引用，还有对语言和话语的讨论，印象深刻。“作为一个河湾”“这也是为什么我特别喜欢幽灵植物的原因之一。它要求你在思考植物的时候，必须超越成本-收益的分析框架。”“我们得用孢子的语言来说话。”用土地说话，用河流行走，用冰雪记录。地质学上，我们只在某纪某世某期的某层，这无法改变。人类像是能穿越时间的幽灵，一根根尖刺突起（物理和语言意义上），钻入本不可破的深时，能看到的只有自身的渺小和孤独。唯能与破溃的当下共处。让自然复凝视我。

《反资本世界简史》
[英国] 大卫·哈维 / 陈诺译 / 广东人民出版社 / 2023
4.17

《思维的发现：关于决策与判断的科学》
[美国] 迈克尔·刘易斯 / 钟莉婷译 / 中信出版社 / 2018
4.21

《跨越式成长》
[美国] 芭芭拉·奥克利 / 汪幼枫译 / 机械工业出版社 / 2020
4.30
