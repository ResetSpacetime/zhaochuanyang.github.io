Nov.10

  • 又饿又冷，无助地被困在被窝里
  • 那些总结归纳，都自有缘由
  • 还是那个自我，得让它的一部分上位
  • “流放者终将归来”
  • “过去不是真实，而是一种技艺，更确切地说，是一种我们借此对过去进行刻画的技艺，一种记忆的技艺。”
  • “重复并不是同样完全相同事物的回归……重复的力量和魅力及带给我们的新奇感，乃是过去事物之可能性的回归。重复能恢复过去的事物的可能性，并使之重新成为可能性。这几乎是一种悖论。”
  • “将时间塞入影像”，灌入，“视觉媒体的本质是时间……”，视界的拓展
  • “影像会凝固起来，变成囚禁人们的幽灵，而人们总是需要从这个幽灵之中重新解放出来……”
  • 「正确的记忆方法是：“记住取出来的方法，而不是记进去。”」
  • “简洁而深刻的解法”
  • u r so use less（From Lili）
  • 大量的通信

Nov.11

  • 购物车基本只剩虾酱、咖喱酱和酒了
  • 还不是人类简史那种想象的共同体，这想象并不奢求共同体验和共情，只要提出另一种选择的自由
  • 吞并在全球各处发生，与之相应的，就是无尽的平静地反讽
  • 简单、抽象领先于时代，（想象的产物），哲学-文学-理性-教育……
  • “每个人都是某物的占有者；所有人都被卷入市场；竞争决定着他们得到什么东西而不得不提供什么东西。”
  • 首先，把一切变成流动的，然后，鼓吹使其自由竞争，最后，分佣提现
  • “没有资本积累就不会有普遍的劳动雇佣”
  • 一无所有者多出现在资本密集地区
  • 小丑准时出源
  • “当然也可能就靠那种特别虚妄的自信，才撑着我干了十几年，就认为自己做的是对的、是正确的，你们就得买我的、读我的书。但是在干的这个过程中，慢慢就知道，能不能别那么觉得自己做的是唯一正向的东西。这是我的一个变化。”
  • 忽然觉得北京人好可怜
  • 简单？哪里有复杂
  • 万物有爱，人恒失之

Nov.12

  • Camper 为啥是从 I.T 电商正式仓发出
  • 资本堆积的弹丸地有什么好怜惜的
  • 自然之传说
  • 问题是抵抗以何种形式发生、持续
  • 停摆的想象
  • “我亦即围绕我的事物”
  • 想象统治术，禁止另一种可能，效率与增长幻象
  • 绿茶（不发酵）、白茶（轻微发酵）、黄茶（轻发酵）、青茶（乌龙茶、半发酵）、黑茶（后发酵）、红茶（全发酵）
  • 失望的终点可能就是，这一切都与我无关
  • 又思及复活节岛，终局自然是悲剧，而它是一个简化的案例：单一文明的物质形态痕迹，任何一个朝代，终局也都不过如此，加速-减速-清零，人生作为“原料”，被大量吞噬，最终完成种种群体想象，留下群体记忆
  • 以自己之力，以自己为原料，达成什么，完成什么过程，接受高峰与低谷，也接受自毁与重来，于是将成就与衰败统一起来
  • 一个人的文明，几个人的文明，一群人的文明，一众人的文明，一国人的文明，拆开看便明朗很多，也清楚它们并不相互从属
  • 若反过来看（摆脱人的视角），则是，想象、目的、神话、技术、文学、语言等等一众产物对人的吞噬
  • 主动或被动地吞噬，应作区分，主动者在完成它和完成它，被动者只是作为食物被吃掉任其肆意
  • 那是自由之守护地吗？不是。
  • 拯救你的脑子，守护好它，摔跤就用力摔，不能怕疼的那种
  • 不要逃掉，不能躲避
  • 不必铭记
  • 历史就是历史，不要扯上“民族”
  • 意义消解就消解，也挺好的

Nov.13

  • 为什么要被闹钟叫醒而不是自然醒，为什么行走要被红绿灯截断/扰乱节奏
  • 远方的声音就是死亡的声音
  • 你说的那些都很有趣，只是与我无关
  • 与我无关很重要，只可做无关之事的观察者，不可以孤立想法为立场，不可以无实之依据二次阐发
  • “汉人做任何事，就是该解决问题的时候打死也要忍着，一味压抑，小事儿越攒越多，越攒越大，简单事儿复杂化，直到越过心理阈限，陷入集体迷狂，火车不撞墙不算完。离远一点看，醉驾式进步，往左打满把，撞墙，包扎修车，往右打满把，包扎修车，无限重复。”
  • “爬虫>哺乳>皮质>思维>思维结构>思想”
  • 规训可能吗？
  • 今天胸变大了吗？有哦

Nov.14

  • “纯粹在精神上的生活是多么冒险”
  • “教育正在扼杀学生的文学理解能力”，“抵抗是一种乐趣”
  • “他的死把我从他身上拔了出来。我一半还活着，一半留着血。”
  • “大家尽量做到自己认可的真实和诚实就行了。有了这样的心理建设之后，我再去做《读库》。有了这个尺度，我尽量按照真实和诚实来做自己想做的事情，当然这个尺度也是随时变化的。”（老六）
  • “我们现在最大的问题是对标准没有共识。可能你我共同欣赏的文字，放在真正跟你利益相关的人那儿，比如给你发工资的老板，反而嗤之以鼻，你怎么办？你还要顺从他的标准，这是问题的症结。……怎样找到和自己标准有共识的人，可能才是最重要的，读库对文字的标准也是动态的，活态的，并且这个标准放到别处肯定不是普适的。我只能说，像今天我们躲在这个小屋里，取得这么一点小小的共识，也就到此为止了。”（老六）
  • “正确性永远没有办法做出判断”，与共识相较，就是有趣之处
  • “大众化的基本逻辑是什么？一对N。我去开一个高端作文培训课，那个N是具体的，1对20，而这个1对N，N可能是1，也可能是一万，可能是一百万，最大的魅力就在于N是不确定的……” （老六）
  • “杜科哈人（蒙古语：Цаатан，Tsaatan）也被称为察坦人或查唐人，是生活在蒙古国库苏古尔省境内一支驯鹿牧民”
  • 努力！吃！
  • “握手楼指的是楼与楼之间的间距不符合国家相关规定，距离太近的楼房。”
  • 不知便不知，理好眼前的盘
