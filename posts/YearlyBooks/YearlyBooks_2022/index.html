2022 阅读

2022-12-16 | 阅读 | 本文共计 1.7k 字 | 阅读约需 6 分钟
Comment

Jan

《欺负人的经济》
[法国] 阿尔贝·雅卡尔 / 新星出版社 / 黄旭颖译 / 2019
1.22 交换。超出利益纠葛。经济的根基。紧密缠绕加速逃离。

《拉比的猫》（全两册）
[法国] 尤安·史法绘著 / 生活·读书·新知三联书店 / 张怡译 / 2016
1.22 布满密集且颇具启发的对话，尤其第一册，逐层拆穿、突破，并非逻辑（异物的假借），仅质疑反问即可，似乎以此打破了很多落后的语言枷锁，打破以语言进行思考时会造成的障碍。另一短评：喜欢上册多一些，思辨密集，怼人甚爽。文明的 bug 无所谓，也并非牢笼，文明和时代所能给予的，无非对话的背景讯息。

Feb

《海怪K》
[意大利] 迪诺·布扎蒂 / 罗国林、吉庆莲译 / 陕西人民出版社 / 1987
2.3 好几篇都非常有趣！短篇，单点的突破，讲述的节奏感，都很精确。对精神与想象的轻柔按摩，混杂着的时代性并不明显。

《叙事的胜利》
[加拿大] 罗伯特·弗尔福德 / 李磊译 / 南京大学出版社 / 2020
2.5 创造叙事与创造新闻，不应混杂

Mar

《幼儿趣味世界地理绘本》
欧洲 意大利 西班牙 希腊 奥地利 葡萄牙 匈牙利 / 明天出版社
3.1

Apr

None.

May

《22款传奇球鞋的前世今生》
黄贺、草威著 / 中国财政经济出版社 / 2022
5.2 读时发现自己穿的就是stan smith…大学时常穿匡威和vans，竟然想试试其他的鞋了。有些庞杂的球鞋世界构建史，很多关键人物（重点没有放在制造和公司层面，果然人类还是更喜欢八卦和故事），工业化和定制化生产让这一些成为可能，关键还是有底层的实际需求（运动与竞技），搭载上亚文化振兴之后，才有机会破了圈。整体仍是一场计算，消费者与品牌并未平等。

《大汖村志》
文化艺术出版社
5.8 有生命力的、亟需记录落档的故事和历史。非常详实的记录，希望再有一百本这样的书。

《我的川菜生活》
石光华 / 陕西师范大学出版社 / 2004
5.14 深入民间的骨髓，深入到热爱生活的个体，不是那么疲于奔命的人，而是陷入在味道中的人，不那么功利和攀比，才觉得更真实。独立于时代的鲜明个体。珍惜食材与探索美味密码，似乎也有时代性。

《间谍过家家（第一卷）》
[日本] 远藤达哉 / 严文瑾译 / 文化发展出版社 / 2020
5.14 看了两本才开始看动画，过于一致了，不知道该放弃哪个。

《解谜茑屋：做未来生活的实验场》
[日本] 川岛蓉子 / 苏暐婷译 / 中信出版集团 / 2019
5.23 果然还是领先国内很多，对需求的洞察和引领，做生活的底色，还有细化服务所在的街区，不知道是通过怎样的内部管理实现的。极早实现了的数字化和档案化，也需要太多现成的社会资源，做不那么费力的创新，需要更深层次的洞察。

Jun

《響 成为小说家的方法》13（完结）
[日本] 柳本光晴 / 小学馆 / 2019
6.12 或许只好这样结尾。向外的探索告一段落，向内的探求刚刚开始。服从所爱，遵照内心。

《碳中和经济学：新约束下的宏观与行业趋势》
中金研究院、中金公司研究部 / 中信出版社 / 2021
6.28 承诺简单，全民共识在哪？谁来承担，如何分工，总是看似想得清楚。

Jul

《纳瓦尔宝典：财富与幸福指南》
[美国] 埃里克·乔根森 / 赵灿译 / 中信出版社 / 2022
7.19 就是多年前那个播客红人…如何想清楚，如何挖掘内心所求，给出了值得参考的共性的答案。沉淀价值，避免被抹除。做出能代表自己坦露自己的产品。

《四大那些事：四大会计师事务所工作生活实录》
啃哥张驰 / 中国市场出版社 / 2015
7.24 看出了一些坚硬的组织管理结构，庞杂的工作，兼顾细节。如此高效、复杂地在为哪个主体服务呢？

《手把手教你读财报：财报是用来排除企业的》
唐朝 / 中国经济出版社 / 2015
7.24 例子有些历史感，专注于欺骗的企业与坦诚的公众公司，时代的产物，终会败给时间。

《如何与重要人物社交》
[美国] 利娅·伯曼、杰里米·伯纳德 / 文家欣译 / 四川人民出版社 / 2020
7.24 奉承的技巧与潜规则。顾及细节，制造惊喜。一种圈子文化中的繁文缛节。似乎他们只能如此运转，无力更新。依传统和文化而活，会更省力一些。

《少来》
韩文斌 / 同济大学出版社 / 2021
7.31 书展二次遇到，那就买了吧。还是本签名本。

Aug

《喜剧的秘密：从脱口秀说起》
贾行家、东东枪、吴琦 / 上海文艺出版社 / 2022
8.16 说不定真是一种草莽创新的路径，玩起来就好。剧场和现场，本就是没什么悬念的行业，要创造的是内容和形式。也是在探索语言本身。

Sep

《难兄难弟:卡尔文与霍布斯虎系列》
[美国] 比尔·沃特森 / 张帆、杨鹏译 / 作家出版社
9.10 总有妙语。不过确实不适合小孩子阅读。

《记忆考古》
长征空间 / 2007
9.10 邱志杰的展览实录。我们真的，好容易遗忘。

《卡尔文与霍布斯虎》
[美国] 比尔·沃特森 / 作家出版社 / 2010
9.25 读完要都做下笔记

Oct

《佛教的见地与修道》
宗萨钦哲仁波切 / 马君美、杨忆祖、陈冠中 / 甘肃民族出版社 / 2006
10.29 不是那么容易的事情。辨析所有概念，认识所有事物，认清所有法门，巨大的工程，需要巨大的虔诚。好在，这条路是明晰的。发掘心的事实。

Nov

None.

Dec

《青之芦苇》1-27卷
小林有吾 / 2022
12.7 希望能借此入门足球（？）

《血之轍》 1-12卷
押見修造 / 小学館 / 2021-11-30
2022-12-10 读过
12.10 阶段性补完。还会有更渺小的展开吧？（男性角度，弱小角度）

《沙村廣明短篇集》SISTER GENERATOR
沙村廣明 / 李昆達译 / 東販
12.11 高跟鞋墓绝了…

《我的单恋比世界危机更重要》
米代恭 / 小学館 / 2016
12.15 不必给男性更多笔墨，太棒了
