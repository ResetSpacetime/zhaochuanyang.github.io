2021 阅读（上）

2021-07-05 | 阅读 | 本文共计 2.5k 字 | 阅读约需 8 分钟
Comment

Jan

《超然物外》
杨威 / 中国社会科学出版社 / 2016
1.8 任意性、抽离、忘我、非知。盖亚、异质、他者、连续与周期。

《末日松茸》
[美国] 罗安清（Anna Tsing） / 华东师范大学出版社 / 2020
1.17 共生关系，复杂的图景与简单的事实。供应链发展的信息非常有趣。对自由联合（团伙作业）之基本准则的探讨。各取所需，因而也成为了整体的一部分。被埋没在资本主义废墟中的历史和生存方式，还有与国家意志的平衡。真是无所不包了。

《秋园》
杨本芬 / 北京联合出版公司 / 2020
1.20 三种文字，一种为正文，一种为后记，一种为遗言。异种的文脉与求生的遗迹，像慢速播放的老实人。苦难转眼就会被遗忘，只剩聚光灯下的起点，和旁边若干个安静的背影。

《1900 年：西方人的叙述 义和团运动亲历者的书信、日记和照片》
[美国] 沙夫 / [英国] 哈林顿 / 顾明译 / 天津人民出版社 / 2010
1.23 其实挺有意思，但信件的信息密度过低，多数书写者也没有什么观察力。

Feb

《创造日本:1853-1964》
[荷兰] 伊恩·布鲁玛 / 倪韬译 / 四川人民出版社 / 2018
2.6 读来轻松，信息量也不低，整合梳理了很多观点，蛮多都在我的认知外。不过更好奇经济发展这条脉络，这个点上信息较少，不过在更宏观的层面给了一些线索（国体、自民党体制、“池田路线”、资金及投资暴涨、出口外向型工业等）。

《战后日本经济史:从喧嚣到沉寂的 70 年》
[日本] 野口悠纪雄 / 张玲译 / 后浪 / 民主与建设出版社 / 2018
2.16 宏观层面了解战后历史的一个视角。政治经济学，及对当下环境与世界格局的理解，及其暗流，都可再加强理解。供应链和日本海外投资部分，仍讲述较少。

《失落中的崛起:日本商业的突围与重生》
陈伟 / 浙江大学出版社 / 2016
2.17 企业史部分太多，重点散乱。提到了一些财阀的信息，可以顺着摸索。

《我的投资生涯:一位日本投资人的自白》
[日本] 村上世彰 / 宋刚、韩涛译 / 机械工业出版社 / 2019
2.21 有深度的价值投资，真的很难，有这样的洞察真的厉害。灵活思考，与时代应和，坚定不移贯彻自己的信念，找到合适的伙伴，这都是成功的关键。资金流转的意义，讲得非常清晰易懂。

《中国溢价:透视中企海外并购》
班妮 / 机械工业出版社 / 2018
2.27 挺好，虽然现在接触不多，但持续的努力与周旋是唯一的办法。口碑与人品一定程度上构成运气。

Mar

《進擊的巨人》
[日本] 諫山創 / 林志昌译 / 東立出版社有限公司 / 2021
3.7 R.I.P. 补至倒数第二话，不断回想起本季动画，孩子如炸弹般从空中掉落

《日本的底力》
徐静波 / 华文出版社 / 2019
3.7 看似杂乱零散，但前半本很多内容都很值得深挖学习。是因写了研究日本商社的短文搜到的，就读了整本，很多地方看得唏嘘，这种埋头做事、潜心研究、积极接轨国际且文化自信的劲头，真的要学习。

《私募股权 LP: 配置策略、投资实践与管理之道》
路跃兵、杨幸鑫 / 中信出版社 / 2017
3.13 不错，内容比较体系化，但实例少了些

《大卫·福斯特·华莱士:最后的访谈》
[美国] 大卫·福斯特·华莱士 / 徐阳子译 / 中信出版集团 / 2019
3.14 相对机智地应付一些无聊的访谈

《金钱不能买什么》
[美国] 迈克尔·桑德尔 / 邓正来译 / 中信出版社 / 2012
3.21 一些例子，一些观察

《医疗投资:基于价值的投资逻辑和实操》
戴广宇 主编 / 机械工业出版社 / 2018
3.22 mindmap

《山中最后一季》
[美国] 埃里克·布雷姆 / 赖盈满、何雨珈译 / 上海社会科学院出版社 / 2016
3.28 以一个人的离去为线索，书写他一生的自然之旅。读来总是有点唏嘘之感。事无巨细地描写他，读来则完全是他对自然的爱和痴迷（哪怕是出轨，或他任何超出理解的决定）。这种复杂的、难于言说的痴迷感，开放的心态，还有止语的留白，真是现世的稀缺物。

《重来3：跳出疯狂的忙碌》
[美国] 贾森·弗里德、[丹麦] 戴维·海涅迈尔·汉森 / 苏西译 / 电子工业出版社 / 2020
3.31 盈利，理性，并保持冷静。什么期望，哪些客户，如何满足，不断向着期望优化并完善。保持沉着与不惊。

Apr

《资本 5000 年：资本秩序如何塑造人类文明》
彭兴庭 / 中国友谊出版公司 / 2021
4.18 太好了！前半本更宏伟些，没这样感受过经济史和工业史。整合了很多信息和观点，难得的视野。很多地方都可以再深入学习下去，是很棒的引子。下面要如何创造未来，我们终于与全球站在了同一起跑线上。要是有个时间线图表就更好了。

《我在硅谷最后一周》
[美国] 科里·佩恩 / 朱天睿译 / 中信出版集团
4.20 不可见的世界。阴谋在很多空间内生发，并膨胀。破溃吗？是的。可惜流动的资本并不惧怕风险，反而肉身和多样性不断被作为燃料。被浪费是很多事情与事物的归宿。数字技术主义就是蔓延的沙漠，就像火星表面（似乎可以被理解为某种终点）。

《副业赚钱》
张丹茹 / 中国友谊出版社 / 2019
4.21 互联网 PM 土鳖多

《反常识管理》
姜海舟 / 中国友谊出版公司 / 2021
4.24 总体够坦诚。不过还要继续续写迭代下去。

《杏仁》
[韩国] 孙元平 / 谢雅玉译 / 民主与建设出版社 / 2019
4.30 结尾有点奇怪。其实是因为0.1工作室的封面才看的。好久没读过虚构，对这种暗结的情节变得陌生。

May

《拥抱可能》
[美国] 伊迪丝·伊娃·埃格尔 / 电子工业出版社 / 2020
5.10 事关改变。Edith 最有发言权，且有最优实践，比大多纯理论的书有说服力。承担责任，承担后果，拥抱生活，问对问题。总有更好的方案，要交流，要开放，要主动寻找，要积极尝试，无惧一定的风险。警惕依赖。

《阿兰的初恋》
[法国] 埃曼努埃尔·吉贝尔 / 崔书凝译 / 后浪、湖南美术出版社 / 2021
5.29 阿兰的讲述总是淡淡的。无法确知的真相，遗憾，还有因未能共担的疾病与灾难而累积筑起的生活。线索，细节，能透露些许，但不容许坦言秘密。

《森林、冰河与鲸》
[日本] 星野道夫 / 曹逸冰译 / 广西师范大学出版社 / 2021
5.30 (装帧五星）没想到可以为鹿之民的阅读再续上一条线索。没有遗憾，聚焦当下。作者似乎也做到了（尽管记录这件事总是略显多余）。这些照片，这些流传着的传说，无非都是契机，人为故事所安排，为氏族之神所安排。消失并不遗憾，重来便是了。

Jun

《蝉》
[澳大利亚] 陈志勇 / 常立译 / 上海人民出版社 / 2019
6.10 另一种结局，非人的结局

《旅行之木》
[日本] 星野道夫 / 曹逸冰译 / 广西师范大学出版社 / 2021
6.14 随笔，信息量一下子就少了些，但混杂乐很多情绪，难以精确言明，于是反复言说，反复尝试，又反复隐灭，归于静谧。读来也反复感受到情绪的波动。孤独之真实，无知之真实，现实之真实，自然之真实。沉湎于想象和逃避过久了，外部的，巨大的，无尽的，理应让人恐惧，找回理应时时相伴的“紧张感”。

《愚比王》
[法国] 阿尔弗雷德·雅里(Alfred Jarry)著 周铭、于硕、方索校 / 中国戏剧出版社 / 2006
6.26 一个世界一种游戏一次永恒的尝试

《墨比斯漫画精选集》
[法国] 墨比斯著 朱洁、叶蔚林、赵心舒译 / 后浪 / 四川文艺出版社 / 2021
6.26 光影。跳跃。联动。很有一种时代感。晚期作品 inside moebius 这种，就只留下极简的思绪的移形换影，一思绪一世界。才能窥见，他的世界是多么具体且周全。

《10人以下小团队管理手册》
[日本] 堀之内克彦著 程雨枫译 / 后浪 / 北京联合出版公司 / 2017
精力稀缺，育人之难，事情交办的艺术，也是沟通的艺术。察言观色很重要，观察细节很重要，还有自然提升的自我要求，因为要做到事出有因，还有表率示范。

《摩根家书》
约翰·皮尔庞特·摩根 / 2012
这种耐心，真的难得一见。信念重要，执行重要，沟通重要，一件一件嵌入在生活和工作中，本就没什么两样，无非自我要求。早早就带领团队，承担责任，公司主体的持续，必然要与稳定和稳固相联系。换个角度，便不再是风雨飘摇，而是成功者的守业和扎根之难。
追评：以这样的能力和眼光，积极搜集整理讯息，继而攫取了整国的财富，真是可怕
